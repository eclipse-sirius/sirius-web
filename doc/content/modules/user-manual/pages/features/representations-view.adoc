= {representations-view}

include::ROOT:partial$before-you-start-experimental-all.adoc[]

Explore the {representations-view} view to gain insights into the available representations associated with the selected element.

image::representations-right-panel-representations.png[{representations-view} view]

This view provides a comprehensive list of all existing representations, offering a convenient overview of the various ways the element is visually represented within the project.

Use it to:

* Open an existing representation in a new tab by clicking its name. If the representation is already open, the editor switches focus to that tab.
* Identify the representation type at a glance—icons and badges help you distinguish diagrams, forms, tables, portals, and more.
* Delete representations via the action available in the view.

TIP: Combine the {representations-view} with the {details-view} to understand both the structure and the available visualizations of an element simultaneously.
